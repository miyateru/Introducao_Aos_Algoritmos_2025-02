# Exemplos de valores

## Valores individuais

1 -> 1*(1+1) / (1+3) = 1*2/4 = 2/4 = 0.5
2 -> 2*(2+1) / (2+3) = 2*3/5 = 6/5 = 1.2
3 -> 3*(3+1) / (3+3) = 3*4/6 = 12/6 = 2
4 -> 4*(4+1) / (4+3) = 4*5/7 = 20/7 ~ 2.857 

## Valores finais

1 = 0.5
2 -> 0.5 + 1.2 = 1.7
3 -> 0.5 + 1.2 + 2 = 3.7
4 -> 0.5 + 1.2 + 2 + 2.857 = 6.557
5 -> 0.5 + 1.2 + 2 + 2.857 + 3.75 = 10.307
